กลุ่มผู้ใช้งาน (Role)

CodeRoleคำอธิบาย
00Defaultกลุ่มผู้ดำเนินการในแบบฟอร์ม (Workflow)

02HQสำหรับ HQ Support
03User Adminให้ User เป็น Admin ระบบ
04Report Adminสำหรับดูเมนูรายงาน / รายงานสรุป

06Report Admin Specialสำหรับเรียกดูเอกสารทั้งหมด
