User Acceptance Criteria: มีการกำหนดสิทธิ์การเข้าถึงข้อมูล — ระบบแสดง/ซ่อนเมนูตาม Role ของผู้ใช้งานที่ Login เข้ามา (Pre-Condition: ผ่านการ login เข้าสู่ระบบ) · จัดการได้เฉพาะ Role 01 Admin และ 02 HQ · ข้อมูลบันทึกที่ตาราง roles / menus / menu_permissions ผ่าน API /api/v1/roles · /api/v1/menus · PUT /api/v1/menu-permissions/{menuCode} และเก็บประวัติที่ audit_logs
