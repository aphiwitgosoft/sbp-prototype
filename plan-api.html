Batch Job scope note: กลุ่ม Batch Job Admin ในหน้านี้เป็น endpoint reference สำหรับ FE Batch Monitor เฉพาะ 2 tab คือ แบบฟอร์มพารามิเตอร์ และ ประวัติการรัน. ไม่ออกแบบ tab Flowchart การทำงาน, งานที่ต้องทำแบบ step-by-step, หรือ Database ที่ใช้ของ batch job ในหน้านี้; รายละเอียดเหล่านั้นให้ดูเอกสาร BE/Runbook/Database reference แยก
